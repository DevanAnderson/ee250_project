External Libraries:
Flask
json
yfinance
grovepi
requests


How to run:
Before running…
For the server:
Type into the terminal, “ pip3 install -r server_requirements.txt” to install the external libraries required
To start the server, type into the terminal “python3 server.py”


For the client:
change the SEVER_URL in client.py to the url of the server. 
Type into the terminal “pip3 install -r client_requirements.txt” to install the external libraries required
To start the client, type into the terminal “python3 client.py”


Make sure to start the server before starting the client.
Multiple clients can connect to the server at once
